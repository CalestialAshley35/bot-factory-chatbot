# Bot Factory Chatbot

Bot Factory Chatbot is a simple yet powerful Discord chatbot maker. With this bot, you can easily create and customize your own Discord bot using simple commands. Whether you're a developer looking to create a personalized bot for your server or just getting started with Discord bot development, Bot Factory Chatbot provides a straightforward way to get your bot up and running.

## Features

- **Greet Users**: The bot can greet users with a friendly message.
- **Ping Command**: Check the bot's latency to Discord.
- **Bot Information**: Get information about the bot, including the author and the number of servers it is in.
- **Invite Link**: Easily generate an invite link to add the bot to other servers.
- **Command List**: Get a list of all available commands.
- **Set Bot Token**: Securely set your bot token to start the bot.
- **Run Bot**: Start the bot after setting the token.
- **Logout**: Log the bot out from Discord.

## Getting Started

To start using Bot Factory Chatbot, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/CalestialAshkey35/bot-factory-chatbot.git
   cd bot-factory-chatbot
   ```

2. **Install the required packages**:
   ```bash
   pip install discord.py
   ```

3. **Edit the bot token**:
   Replace `'YOUR_BOT_TOKEN_HERE'` in the `bot.run` method with your actual bot token.

4. **Run the bot**:
   ```bash
   python bot_factory_chatbot.py
   ```

Alternatively, you can use the online tool provided to create and customize your bot without needing to set up anything locally.

## Usage

Once the bot is running, you can interact with it using the following commands:

- `!hello` - Greets you with a friendly message.
- `!ping` - Checks the bot's latency to Discord.
- `!info` - Provides information about the bot.
- `!invite` - Provides an invite link for adding the bot to your server.
- `!commands` - Lists all available commands.
- `!set_token <your_bot_token>` - Sets the bot token.
- `!run_bot` - Starts the bot after the token is set.
- `!logout` - Logs the bot out from Discord.
